### Quick start

**Install**

```coffee
install.packages("devtools")
library("devtools")
install_github("ropensci/enigma")
library("enigma")
```

**Search for datasets**

```coffee
print("foo bar")
```

**Get data**

```coffee
print("foo bar")
```

**Get statistics on datasets**

